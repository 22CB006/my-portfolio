# Pages Directory

## Pages

- **Home.js** - Landing page with all sections
- **Projects.js** - Projects listing with filters
- **ProjectDetail.js** - Individual project case study
- **Experience.js** - Work experience timeline
- **About.js** - Professional profile and skills
- **Contact.js** - Contact information and form
- **NotFound.js** - 404 error page

## Conventions

- Each page is a top-level route component
- Pages compose smaller components from /components
- Pages load data from /data files
- Use React Router hooks for navigation and params
