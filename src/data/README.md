# Data Directory

## Data Files

- **projects.js** - Project data with schema and sample projects
- **experience.js** - Work experience data
- **skills.js** - Skill groups and categories
- **education.js** - Education information
- **certifications.js** - Certifications list
- **config.js** - Site configuration (name, title, contact info, navigation)

## Conventions

- Export data as named exports
- Use consistent data schemas across all entries
- Keep data separate from UI components
- All content updates should happen in these files
